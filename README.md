# Crypto Pattern Predictor Bot (1H Candle Pattern Matching)

این پروژه یک ربات تلگرام است که با استفاده از الگوی ۹ کندل گذشته،  
در سایر ارزها الگوهای مشابه را پیدا کرده و **کندل ۶ را به عنوان پیش‌بینی کندل ۰** ارائه می‌دهد.

## ✨ ویژگی‌ها
- دریافت کندل‌ها از CoinEx و BingX
- تحلیل فقط در تایم‌فریم ۱ ساعته
- مقایسه الگوهای ۱۰ کندلی
- پیش‌بینی کندل ۰ بر اساس رفتار کندل ۶
- Webhook کامل با FastAPI
- آماده‌ی دیپلوی روی Render
